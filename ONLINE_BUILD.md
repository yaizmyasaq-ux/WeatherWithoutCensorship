# Сборка APK через GitHub

1. Создайте новый репозиторий на GitHub.
2. Распакуйте архив проекта.
3. Загрузите в репозиторий содержимое папки `WeatherWithoutCensorship`, включая скрытую папку `.github`.
4. Откройте вкладку **Actions**.
5. Выберите **Build Android APK**.
6. Нажмите **Run workflow** → **Run workflow**.
7. После зелёной галочки откройте завершённый запуск.
8. В разделе **Artifacts** скачайте `weather-without-censorship-apk`.
9. Распакуйте скачанный ZIP. Внутри находится `app-debug.apk`.

Для установки APK на телефоне разрешите браузеру или файловому менеджеру установку приложений из неизвестных источников.
