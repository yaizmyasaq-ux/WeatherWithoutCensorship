# Погода без цензуры — Android-виджет

Минимальный виджет 2×1 для рабочего стола Android:

- показывает текущую температуру;
- учитывает температуру «ощущается как» при выборе фразы;
- выводит погодный эмодзи;
- обновляется примерно раз в 30 минут;
- имеет ручное обновление по кнопке ↻;
- использует Open-Meteo, API-ключ не нужен.

## Сборка

1. Открыть папку проекта в Android Studio.
2. Дождаться Gradle Sync.
3. Подключить Android-телефон с включённой USB-отладкой либо выбрать эмулятор.
4. Нажать Run.
5. Открыть приложение и разрешить геопозицию.
6. На рабочем столе удержать пустое место → «Виджеты» → «Погода без цензуры».

## Где менять фразы

`app/src/main/java/com/houston/weatherwidget/WeatherWorker.java`

Метод `phraseFor(int temperature)`.

## Ограничения

- Android сам выбирает точное время фонового запуска, поэтому 30 минут — ориентир.
- После переезда в другой город открой приложение и снова нажми «Определить место».
- При запрете фоновой работы оболочкой телефона обновления могут задерживаться.
